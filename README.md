# React Todo List with Hooks

Це практична робота на основі попередньої роботи "Бібліотека React". Мета: реалізувати редагування списку задач із застосуванням **React Hooks**.

## Функціонал
- Додавання задач
- редагування задач
- Видалення задач
- помічення вже виконаних задач
- Відображення списку задач у реальному часі

## Використані технології
- React
- React Hooks (`useState`)
- JavaScript, HTML, CSS
- Git/GitHub для зберігання коду
